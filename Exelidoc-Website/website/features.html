Why Exelidoc Is Different

The freedom to choose: local privacy or cloud power

🖥️ Local AI — Your Computer, Your Data

With Local AI powered by Ollama, everything runs on your computer. Your documents never leave your machine.

Perfect For:

  • Confidential documents (contracts, medical records, legal)
  • Privacy-conscious professionals
  • Companies with data compliance requirements
  • Users without reliable internet
  • Zero API costs

How It Works

You download Ollama (open-source AI framework), run a model on your machine, and Exelidoc connects to it. Everything stays local. No cloud. No third parties. No data leaving your computer.

Getting Started with Local AI

  1. Download Ollama from ollama.com
  2. Run: ollama pull llama3
  3. Run: ollama serve
  4. In Exelidoc, switch to "Local AI" mode
  5. Start using — no API costs, no internet needed

💼 Example Use Case

A lawyer reviewing confidential client documents. Uses Local AI so the document text never leaves their computer. Complete privacy, no API costs.

☁️ Cloud AI — Maximum Power & Speed

Connect to state-of-the-art AI models from OpenAI (GPT), Anthropic (Claude), or Google (Gemini). Get the best possible responses in seconds.

Perfect For:

  • Fastest, most powerful responses
  • Latest AI models (GPT-4, Claude, Gemini)
  • Professional writers & researchers
  • Content creators
  • Pay-as-you-go (very affordable)

How It Works

You provide your API key (OpenAI, Anthropic, or Gemini). Exelidoc sends requests to their servers and gets back the best AI responses available. You pay only for the tokens you use.

Typical Cloud AI Costs

  • GPT-4: ~$0.03 per 1,000 tokens
  • Claude 3: ~$0.003 per 1,000 tokens (very cheap)
  • Gemini: ~$0.0005 per 1,000 tokens (cheapest)

Translation: You can get 1 million tokens for $0.50–$30, depending on the model.

📝 Example Use Case

A content creator writing blog articles. Uses Cloud AI with Claude for high-quality writing. Costs ~$5/month. Gets published-ready content in seconds.

Which One Should You Choose?

✅ Choose Local AI If:

• You're working with confidential documents
• Privacy is your top priority
• You don't want to pay per request
• You're okay with slightly slower responses
• You have a decent computer

✅ Choose Cloud AI If:

• You want the best possible AI responses
• Speed matters to you
• You want GPT-4 or Claude quality
• You're fine with minimal API costs ($5-20/month)
• You don't want to install extra software

🎯 Best Part: You can switch between them anytime. Use local for sensitive docs, cloud for everything else.

Frequently Asked Questions

What's Ollama?

Ollama is a free, open-source framework for running AI models locally. It handles all the technical complexity. You just download it and click a button.

Do I need to be tech-savvy to use Local AI?

No. Just download Ollama and run it. Exelidoc handles the rest. It's as easy as downloading Chrome.

Is Cloud AI expensive?

No. Claude costs about $0.003 per 1,000 tokens. You can write 100,000 words for $0.30. Most users spend $5-20/month.

Can I switch between Local and Cloud?

Yes! One-click toggle in Exelidoc. Use local for sensitive docs, cloud for everything else.
