Getting Started in 5 Minutes

Download, install, and start using Exelidoc

What You'll Learn

This guide walks you through:

  • Downloading Exelidoc
  • Running the launcher app
  • Loading the add-in into Word/Excel/PowerPoint
  • Choosing Cloud or Local AI
  • Making your first request

1Download Exelidoc

Go to the download page and click "Download Now"

The file is about 150 MB. It includes everything you need.

💡 Tip: Save it somewhere you'll remember (Desktop or Documents folder is fine)

2Run the Installer

After downloading, double-click the installer (or exe file)

[Installer window appears]

Click "Install" → Exelidoc gets installed to your computer

On Windows, you might see a security warning. Click "Run" to proceed.

✓ Done! A desktop icon appears. Click it anytime to launch Exelidoc.

3Launch Exelidoc & Configure

Double-click the Exelidoc icon on your desktop

The launcher window opens

You see: Settings tab, Server tab, Help tab

Choose Your AI Mode:

Setup Cloud AI (2 minutes)

1. Get an API key:

  • Go to console.anthropic.com (Claude - recommended, cheapest)
  • Sign up → copy your API key

2. Paste the key in Exelidoc:

  1. In the launcher, go to "Settings" tab
  2. Paste your API key in the "Anthropic Key" field
  3. Click "Save"

3. Start the server:

  1. Click "Start Server" button
  2. Status shows "Server running at http://localhost:8000"
✓ Cloud AI is ready! Move to Step 4.

Setup Local AI (5 minutes)

1. Download Ollama:

  • Go to ollama.com
  • Download and install
  • Open Command Prompt or Terminal

2. Download a model:

ollama pull llama3

This downloads a 4GB AI model (first time only, takes a few minutes)

3. Start Ollama:

ollama serve

Keep this window open. Your computer is now running local AI.

4. In Exelidoc, click "Start Server"

Exelidoc will auto-detect Ollama and connect to it.

✓ Local AI is ready! Move to Step 4. No API costs, complete privacy.

4Load Into Office

Now you'll tell Word/Excel/PowerPoint where to find Exelidoc.

In Word (same for Excel/PowerPoint):

  1. Open Word
  2. Go to Insert menu
  3. Click Add-ins → Get Add-ins
  4. Click Upload My Add-in
  5. Navigate to the manifest.xml file (in your Exelidoc folder)
  6. Click Upload
[Right sidebar shows Exelidoc panel]

The Exelidoc panel appears in your right sidebar

✓ Exelidoc is now in Word! Do the same for Excel and PowerPoint if you want.

5Make Your First Request

Try it:

  1. In the Exelidoc panel, type a question (e.g., "Summarize this document")
  2. Click "Send"
  3. Watch the magic happen! ✨
💡 Pro Tips:
  • Use the "Tools" tab for one-click actions (Summarize, Fix Grammar, etc.)
  • Click "Use Selection" to pull highlighted text into your question
  • Watch your token counter in the Account tab
🎉 You're done! You're now using Exelidoc.

Troubleshooting

🔴 "Server is offline" or won't start

Cloud AI: Did you paste your API key in Settings? Try clicking "Start Server" again.

Local AI: Is Ollama running? Open Terminal/Command Prompt and run ollama serve

🔴 The add-in won't load in Office

Is the Exelidoc server running (green status in launcher)? Try hard-refreshing Office with Ctrl+F5.

🔴 "API key is invalid"

Make sure you copied your key correctly from console.anthropic.com

🔴 Everything is frozen/slow

If using Local AI, it depends on your computer's speed. Local AI is slower than cloud.
